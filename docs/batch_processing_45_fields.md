## Обзор

## Входной формат

CSV или XLSX с колонками (поддерживаются русские и английские названия):

- **ФИО** (обязательно) — ФИО должника
- **Адрес** (обязательно) — адрес регистрации
- Паспорт — серия и номер
- Сумма / debt_amount — сумма задолженности (для расчёта госпошлины)
- Широта / lat — координата (если есть)
- Долгота / lng — координата (если есть)

## API

### POST /api/v1/batch-process

Загрузка файла CSV или XLSX.

**Параметры:**
- `file` — файл (обязательно)
- `format` — `json` (по умолчанию) или `xlsx`

**Ответ при format=json:**
```json
{
  "success": true,
  "total": 10,
  "ok": 8,
  "errors": 2,
  "elapsed_ms": 2500,
  "results": [...]
}
```

**Ответ при format=xlsx:** файл XLSX для скачивания (листы: Результаты, Статистика, Примечания).

## Использование в коде

```python
from batch_processing import process_debtor, process_batch
from batch_processing.utils.file_handler import read_file
from batch_processing.services.output_generator import generate_xlsx

# Один должник
r = process_debtor(
    fio="Иванов И.И.",
    address="Москва, ул. Тверская 1",
    debt_amount=15000,
)

# Батч из файла
debtors = read_file(Path("data.csv"))
results = process_batch(debtors)
generate_xlsx(results, Path("output.xlsx"))
```

## Пакетная обработка по GPS (без парсинга адресов)

**Эндпоинт:** `POST /api/v1/batch-gps-process`

Работа напрямую с координатами — без парсинга и геокодирования адресов.

**Поддерживаемые форматы:** CSV, XLSX, GeoJSON, KML

**Обязательные колонки:** `lat` (широта), `lon`/`lng` (долгота)  
**Опционально:** `case_type` (тип дела), `debt_amount` (сумма для госпошлины)

**Пример CSV:**
```
lat;lon;case_type;debt_amount
55.7558;37.6173;Гражданское;15000
59.9343;30.3351;Административное;20000
```

**Пример GeoJSON:**
```json
{
  "type": "FeatureCollection",
  "features": [{
    "type": "Feature",
    "geometry": {"type": "Point", "coordinates": [37.6173, 55.7558]},
    "properties": {"case_type": "Гражданское", "debt_amount": 15000}
  }]
}
```

**Тест:**
```bash
python scripts/run_batch_gps_test.py
```
